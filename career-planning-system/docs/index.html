项目背景

当前大学生在职业规划中普遍面临自我认知模糊、岗位信息不对称、个性化指导缺失三大核心痛点。本项目以服务外包创新模式，依托AI与数据驱动技术，构建专业化智能职业规划系统，引导学生从模糊的职业意向走向清晰化、可执行、可评估的职业发展闭环。

核心功能

覆盖"岗位认知 → 自我认知 → 匹配分析 → 行动规划"的完整闭环

1

岗位画像与关联图谱

基于 10,000+ 真实招聘数据，AI聚合分析生成计算机领域15个核心岗位的精细化画像（技能、证书、薪资等）。配备垂直晋升路径和横向换岗路径两类图谱，清晰呈现职业发展空间。

2

七维能力画像

支持 PDF/Word 简历上传，AI自动解析并生成覆盖专业技能、证书资质、创新能力、学习能力、抗压能力、沟通能力、实习经历七个维度的量化能力画像，输出综合竞争力评分、能力短板及优先级改进建议。

3

四维人岗匹配

从基础要求（25%）、职业技能（35%）、职业素养（25%）、发展潜力（15%）四个维度进行加权匹配评估，精确对比学生能力画像与目标岗位要求，量化匹配度并定位技能差距。匹配准确率达82%。

4

个性化职业规划报告

自动生成包含人岗匹配分析、职业目标设定、分阶段行动计划、风险评估的结构化报告。支持在线编辑与一键PDF导出，推动规划落地执行。采用异步生成+进度轮询机制优化用户体验。

核心技术亮点

多项技术创新保障系统的准确性与可靠性

1

多级降级 RAG 检索增强可信体系

构建"Qdrant向量检索 → 本地向量表 → 结构化知识库"三级检索降级架构。768维稠密向量编码+HNSW索引实现毫秒级语义检索，召回准确率92%+。当向量库异常时自动平滑降级，服务可用性达99.9%，从根源抑制AI幻觉问题。

2

LangGraph 状态机式报告生成引擎

引入LangChain+LangGraph状态机框架，将报告生成分解为自我评估、职业路径、技能差距、行动计划、风险评估五个阶段，每个阶段由独立Prompt节点+Few-shot示例驱动，结构完整率达98%，支持实时进度反馈。

3

四维加权人岗匹配算法

从基础要求、职业技能、职业素养、发展潜力四个维度，结合AI精确逐项对比算法进行加权评分。支持匹配技能与缺失技能的精准定位及差距优先级排序，匹配准确率达82%。

4

多格式简历智能解析

支持PDF/Word(docx/doc)三种主流格式，组合python-docx2txt+pdfplumber+LibreOffice实现文本提取，结合大模型Few-shot学习完成12项核心字段结构化抽取，为能力画像提供全面数据基础。
